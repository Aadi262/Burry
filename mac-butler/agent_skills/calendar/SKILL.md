# Calendar Lookup

Use this skill for schedule, event, and reminder requests.

1. Prefer the existing calendar-related Burry tools or skills.
2. When the user asks for a reminder, use `set_reminder`.
3. When the user asks what is on the calendar, answer from the calendar skill output.
4. Keep the spoken reply focused on the relevant events or reminder outcome.
